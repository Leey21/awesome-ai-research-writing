# 概念图设计技能（canvas-design）

> 来源：[anthropics/skills](https://github.com/anthropics/skills)
> 安装：`npx openskills install anthropics/skills`

## 功能介绍

先产出 design philosophy (.md)，再在画布上实现为单页 .png / .pdf，适合论文中的：
- 概念图
- 示意图
- 框架图（整体 pipeline）

可与 `论文写作技能` 配合使用，生成图后直接用 caption 规范写说明。

---

## 使用场景与示例

### 论文概念图 / 示意图 / 框架图

**前置输入**：图的用途与大致元素（如三阶段 pipeline、方法对比）

**示例 Prompt**：
- 「帮我画一个我们方法的整体框架图，三块：数据、训练、推理」
- 「做一张方法对比的示意图，左边传统方法，右边我们的」

**产出**：design philosophy (.md) + 可下载的 .pdf 或 .png，可插入 LaTeX 并配合论文写作技能写 caption

---

### 修改图的风格或细节

**前置输入**：对已有图的修改意见

**示例 Prompt**：
- 「背景改成浅灰」
- 「左边块加大一点」
- 「不要那么多字，只保留标签」

**产出**：按意见调整后的新版图说明，再导出 .pdf/.png 供替换进论文
