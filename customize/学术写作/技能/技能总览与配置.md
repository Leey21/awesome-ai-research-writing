# 论文写作 Agent Skills 总览与配置

> 适用对象：经常使用 Cursor、Claude Code 等 AI coding 工具的用户
>
> Agent Skills 是一种可被 AI 助手（如 Claude、Cursor）加载的扩展能力包，内含针对特定任务的流程、规范与模板。配置后，在对话中直接描述需求即可触发对应流程，无需记忆复杂 prompt。

---

## 环境依赖

- Node.js 20.6+（含 npm）
- Git

---

## 安装 OpenSkills

```bash
# 临时使用（无需安装）
npx openskills --version

# 全局安装（多项目复用）
npm i -g openskills
openskills --version
```

---

## 一键安装学术写作 Skills

```bash
# 安装学术研究相关 Skills
npx openskills install zechenzhangAGI/AI-research-SKILLs

# 安装 Anthropic 官方 Skills
npx openskills install anthropics/skills
```

执行后会弹出交互式选择，勾选需要的 Skill 即可（默认全部安装）。

---

## 在 Cursor 中验证配置

1. **确认安装**：`npx openskills list` 查看已安装 skills
2. **Settings 中查看**：Cursor Settings → Rules, Skills, Subagents → Skills 区域
3. **手动调用**：Agent Chat 输入 `/`，搜索 skill 名称
4. **自然触发**：直接说明需求（如"用 ICLR 2026 模板新建一篇论文"）

## Skills 总览

| Skill 名称 | 来源 | 功能简述 |
|------------|------|----------|
| **论文写作**（20-ml-paper-writing） | [zechenzhangAGI/AI-research-SKILLs](https://github.com/zechenzhangAGI/AI-research-SKILLs) | 面向 NeurIPS/ICML/ICLR/ACL/AAAI/COLM 的完整论文写作：从 repo 起稿、LaTeX 模板、引用验证、审稿人视角、会议 checklist、格式迁移；内含 booktabs 表格规范与图规范。 |
| **去AI味**（humanizer） | [blader/humanizer](https://github.com/blader/humanizer) | 识别并去除 AI 写作痕迹，使文本更自然、像人写。基于 Wikipedia「Signs of AI writing」，同时注入「人味」：有观点、节奏变化、承认不确定性。 |
| **Word文档**（docx） | [anthropics/skills](https://github.com/anthropics/skills) | 对 .docx 进行创建、编辑、分析。支持用 Word 模板生成投稿稿，也可做带修订痕迹的审稿式修改。 |
| **协作写作**（doc-coauthoring） | [anthropics/skills](https://github.com/anthropics/skills) | 分阶段文档协作：收集上下文与澄清问题 → 按节头脑风暴 → 起草 → 精修 → 读者测试查盲点。 |
| **概念图设计**（canvas-design） | [anthropics/skills](https://github.com/anthropics/skills) | 先产出 design philosophy (.md)，再在画布上实现为单页 .png / .pdf，适合论文中的概念图、示意图、框架图。 |
