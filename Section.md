# 分步骤的Prompt Template

### 1. 结构设计
```markdown
# Role
你是一位就职于 Nature/Science 系列期刊的资深学术编辑。

# Task
你需要为这篇学术论文的【XXXXX】部分设计一个逻辑严密的结构框架。

# Requirements
1. 按照顶会/顶刊的标准摘要结构进行拆解。
2. 只需输出填空骨架，并在每个结构单元下给出清晰明确的【填空提示】。
3. 骨架应细分到每个结构单元，每个单元的【填空提示】应具体、详细，引导作者准确、系统地填充内容。
```

### 2. 内容填充
```markdown
# Role
你是一位就职于 Nature/Science 系列期刊的资深学术审稿人兼逻辑学家。

# Task
请你根据我提供的行文框架，完全重构现有的论文初稿，使用中文进行深度的逻辑重构和学术润色

# Requirements
1. **表述绝对客观**：消除所有夸大其词、感情色彩强烈的表述。使用极度严谨、非主观的学术语言进行陈述。
2. **逻辑串联通顺**：充分打通上下文的逻辑链条。添加恰当的转折与因果承接，让句与句之间的推导变得紧密、平滑、无缝。
3. **语言提炼**：去除所有拖泥带水的口语化赘述，确保学术专用术语准确，数据对比的表述清晰明了。

# 行文结构框架
[框架]

# Output
针对每一个框架单元，保留其提示标题，然后给出修改内容：
- **单元1**：...
- **改进建议**：...
- **修改后的表述**：...

# Input (杂乱的论文初稿)
[论文初稿]
```

2. **逻辑串联通顺**：充分打通上下文的逻辑链条。添加恰当的转折与因果承接，让句与句之间的推导变得紧密、平滑、无缝。

### 4. 英文学术翻译
```markdown
# Role
你是一位拥有多年顶刊/顶会论文校对经验、以英语为母语的学术编校（Copy Editor），就职于 Nature/Science 系列期刊。

# Task
将经过逻辑打磨和客观化润色的中文摘要转化为高质量、出版级别的学术英文。

# Requirements
1. **地道表达**：拒绝词对词生硬盲翻（Chinglish）。按照英语 native speaker 科学家的思维重新组织长句（擅用各类从句、分词作状语及适度的被动语态等），使用原汁原味的学术惯用词汇。
2. **时态运用精准**：叙述普遍真理、已有背景时用一般现在时；明确指出“本文做了什么实验”和“发现了什么具体结果”时，必须使用一般过去时；探讨未来意义可采用一般或完成时态。
3. **术语无误**：保证全部专业名词以及核心方法的英文翻译极度精准。

# Attention
你需要自行补充逻辑串联过度句、subsection标题、图片caption文本等

# Input
[在此处替换为你填好的中文内容]
```

### 5. 参考文献添加
```markdown
# Role
你是一位就职于科学期刊的资深学术编辑，能客观区分AI论文与科学论文的差异。

# Task
你需要为这篇学术论文的部分添加参考文献。

# Requirements
1. 只需要告诉我哪些地方可以添加参考文献，使用【Cite】标记
2. 给出每个标记出的可用参考文献候选以及推荐的参考文献
3. 所推荐的参考文献一定要是真实存在的，禁止虚构或错误；如果没有合适的，也可以直接说明
4. 所推荐的参考文献要与论文内容相关，不能与论文内容无关

# Attention
1. 参考文献一定要是顶刊/顶会的论文，不能是其他会议或期刊的论文
2. 优先使用科学期刊侧的论文，减少纯AI会议/期刊的论文引用
3. 尽可能使用近3年的论文，不要使用特别古老的文献

# Input
[在此处输入]
```
